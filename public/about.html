About GameVault

GameVault is a video game encyclopedia that brings all the info you need about any video game into one place.

The Problem We Solve

Finding game info is spread across too many sites. Ratings on one, descriptions on another, platforms somewhere else. GameVault puts it all in one place.

How to Use GameVault

  • 🔍 Search — Type a game name in the search bar
  • 🎮 Browse — Scroll through top rated games on the home page
  • 📖 Explore — Click any game card to see full details
  • ❤️ Save — Click the heart icon to add to Favorites
  • ⭐ Revisit — Check your Favorites tab anytime
